My Starter Kit

Description

A simple kit to get you started. Don't worry about a damn thing, let Gulp take care of the build process.

System Preparation

To use this project, you'll need the following things installed on your machine.

  • NodeJS - use the installer
  • Gulp - $ npm install -g gulp

Local Installation

  • Clone or fork this repo, or download it into a directory of your choice.
  • Inside the directory, run npm install.

Usage

DEVELOPMENT MODE

Jus type gulp into the terminal of the root of your project. A browser will open and you can view your project while you work and watch the changes happen as you edit and save you files. 

$ gulp
